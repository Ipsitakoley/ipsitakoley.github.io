Term Project

Project 2: AUTOSAR compliant Controller

Description and Objective

The objective of this project is to implement an AUTOSAR (AUTomotive Open System Architecture) compliant controller in Infineon Tricore Board. AUTOSAR standard is widely adopted OEMS and suppliers in the automotive industry. Implement an existing trajectory/cruise control application with ETAS- AUTOSAR tools - RT-BSW, RTA-OS, RTA-RTE and Infineon tricore TC2xx Board. The plant model will be running in HIL setup-LABCAR and COSYM environment will be used to model and monitor the plant. The controller and HIL will interact with each other through CAN.
